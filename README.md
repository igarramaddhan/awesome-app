##Setup React Project from Scratch

#### Run Locally

1. **Clone**
    ```sh
    git clone https://github.com/igarramaddhan/awesome-app.git
    ```

1.  **Install dependency**

    Navigate into the cloned directory
    ```sh
    cd awesome-app/
    yarn
    ```

1.  **Start the server**
    ```sh
    yarn dev
    ```
